# webpack-4-examples

Examples of various webpack configs, generally increasing in complexity.

For each subdirectory:

- `npm install`

For 01:

- `./node_modules/.bin/webpack`

For 02 - 03:
- `./node_modules/.bin/webpack --config=webpack.config.js`

For 04-14: 

- `npm start` (examples 04 - 14)
